# About

Plugins for [Seanime](https://github.com/5rahim/seanime). Read the official docs for the plugin [here](https://seanime.gitbook.io/seanime-extensions/)

## Dev

First run `pnpm update-types` to get the type definitions needed for the plugins.

To serve the plugin locally you can do

```bash
pnpm run dev
```

You can then test it by importing to your local seanime instance.

Alternatively, you can follow the instruction from the [docs](https://seanime.gitbook.io/seanime-extensions/plugins/write-test-share#test-it-live)
